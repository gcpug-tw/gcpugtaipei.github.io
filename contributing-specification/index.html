投搞指南

GDGCloud Taipei 透過網際網路串聯喜好 Google Cloud 的使用者社群，Facebook GCPUG.TW 群組上都會有朋友分享相關 Google Cloud Platfrom 相關的文章，不過因為 Facebook 群組的機制並不適合保存這些文章，所以希望可以透過一個平台來收錄相關的文章

投稿文章的分類

  • 原創：管理員會對投稿文章質量做一定的考察，必要的情況下，作者需要聯繫社群管理員提供相關個人背景信息；對於長期投稿或知名度較高的作者，我們將邀請成為社群的特邀作者，審核和評審流程時間會相應減短。

  • 翻譯：鼓勵多提供高質量文章線索，管理員會簡要瀏覽全文考察文章質量和討論的話題，辨別並過濾掉低質量文章和廣告軟文；另外，管理員會盡量同原文作者取得聯繫，通知作者我們翻譯了他的文章；避免翻譯來自其它社群或科技媒體發表的文章，除非獲得官方授權。

  • 轉載：個人博客博主和公眾號可以對我們的微信公眾號開通白名單，管理員會定期挑選轉載相關文章；也可以通過PR提出相關文章轉載請求。
